🎯 LA NUEVA FUNCIÓN (sendQRToHikvision)
════════════════════════════════════════════════════════════════════════════

¿Qué hace?
│
├─ ENTRADA: token (8 dígitos), visitorName (nombre)
│
├─ PASO 1: Registra usuario en Hikvision
│   POST /ISAPI/AccessControl/UserInfo/Record
│   Resultado: Usuario creado con permisos
│
├─ PASO 2: Registra tarjeta virtual (QR)
│   POST /ISAPI/AccessControl/CardInfo/Record
│   Resultado: Tarjeta lista para escanear
│
├─ PASO 3: Genera imagen QR
│   qrcode.toDataURL(token)
│   Resultado: data:image/png;base64,...
│
└─ SALIDA: Objeto con { token, qrImage, cardNo }
